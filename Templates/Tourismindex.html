About W3Schools


We create simplified and interactive learning experiences. Learning web development should be easy to understand and available for everyone, everywhere!
W3Schools is a school for web developers, covering all the aspects of web development: HTML Tutorial. CSS Tutorial. JavaScript Tutorial.
It is run by Refsnes Data in Norway. It has an online text editor called TryIt Editor, and readers can edit examples and run the code in a test environment.
